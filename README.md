# YMF825Board

## Directory
### manual
IF Specification of YMF825Board

### sample1
Sample Program that generates "Do-Re-Mi-Fa-So" automatically on Arduino Uno

### sample2
Software Module that makes YMF825 a MIDI tone generator written in C Language

### tonedata
Tone data of YMF825
